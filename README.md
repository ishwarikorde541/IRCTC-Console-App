# IRCTC-Console-App

A Java-based console application that simulates basic IRCTC functionalities.

## Features
- User Registration
- User Login
- Train Search
- Ticket Booking
- Ticket Cancellation
- View Booked Tickets
- Train Management

## Technologies Used
- Java
- OOP Concepts
- Collections Framework
- LocalDate API

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run IRCTCAPP.java
